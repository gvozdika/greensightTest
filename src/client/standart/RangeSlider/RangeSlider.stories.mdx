# RangeSlider

Компонент `RangeSlider` использует input type="number" и компонент `Range` взятый из [rc-slider](https://github.com/react-component/slider). Предназначен для вывода и установки диапазона значений. Внутри компонента производится пересчет введенных в поля значений. Для обновления состояний у отрендеренного компонента необходимо изменить values.
